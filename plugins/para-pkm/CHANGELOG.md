# Changelog

All notable changes to the PARA PKM plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2025-12-21

### Added

- Initial release of PARA PKM plugin
- Complete PARA knowledge management skill
- Python scripts for automation:
  - `init_para_kb.py` - Initialize new PARA knowledge base
  - `validate_para.py` - Validate PARA structure
  - `archive_project.py` - Archive completed projects
  - `generate_nav.py` - Generate AI-friendly navigation
- Templates for projects, areas, and navigation files
- Comprehensive reference documentation:
  - PARA principles and methodology
  - Decision guide for categorizing content
  - Common patterns by role (developers, consultants, researchers, product builders)
  - AI navigation best practices
- Assets including templates for:
  - Project files
  - Area overview files
  - AI agent navigation (AGENTS.md)
  - Knowledge base README
- Full integration with Claude Code plugin system

### Documentation

- Comprehensive README with quick start guide
- Examples for different use cases and roles
- Decision trees for content categorization
- Anti-patterns to avoid
- Troubleshooting guide
