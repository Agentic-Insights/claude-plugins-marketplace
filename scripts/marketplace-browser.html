Agent Skills Specification
Required Fields
All skills must include these fields in SKILL.md frontmatter:
name
Unique identifier for the skill. Must be lowercase with hyphens only.
name: my-awesome-skill
description
Clear, actionable description of what the skill does and when to use it. 10-1024 characters.
description: "Validates TypeScript code and suggests optimizations"
Optional Fields
Enhance your skill metadata with these optional fields:
license
SPDX license identifier for the skill.
license: Apache-2.0
compatibility
Environment and dependency requirements.
compatibility: "Python 3.9+, Node.js 16+, Docker"
metadata
Custom key-value pairs for additional context.
metadata: author: "Your Name" version: "1.0.0" tags: [tag1, tag2]
allowed-tools
Pre-approved tools for sandboxed execution.
allowed-tools: "bash python node"
